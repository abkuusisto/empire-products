What we check and what we do not

What StoreGuard checks
pass - Homepage reachability and response time
pass - HTTPS/SSL connection
pass - Platform detection
pass - Cart path HTTP reachability
pass - Checkout signals in page HTML
pass - Payment trust signals
pass - Refund and shipping content
What StoreGuard does not check
no - Actual checkout completion or payment processing
no - JavaScript-rendered content
no - Login-gated or region-gated pages
no - Real-time inventory or pricing accuracy
no - Mobile vs desktop rendering
Scan scope disclosureStoreGuard uses a lightweight read-only HTTP/HTML scan. No JavaScript is executed. No transactions are initiated. No items are added to cart. No payments are processed. Findings are informational only.
